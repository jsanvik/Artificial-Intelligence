DFS but: 
- assign 1 variable at a time
    - Most constrained variable: assign variable with the fewest legal values left in its domain
    - Least constraining value: assign variable to value that rules out the fewest values in the remaining variables
- check constraints at each step
- Cross off values that violate a constraint when added to existing assignment
    - If X loses a value, X's neighbours need to be checked!
