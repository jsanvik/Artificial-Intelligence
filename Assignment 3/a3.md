### 3. [15 points]
**Assume the following tree-structured CSP, where you have 6 regions that should be colored using 2 different colors given no 2 neighboring regions with the same color. How the algorithm for tree-stuctured CSPs will solve this problem? Explain.**  

**Variables:** Regions 1, 2, 3, 4, 5, 6  
**Domains:** {Colour 1, Colour 2}  
**Successor Function:** Assign a colour to each region  
**Constraints:**  
- No region is the same colour as a neighbouring region  
**Goal Test:** All assignments satisfy constraints  

The problem must be structured as a graph, where each node is a region and an edge connects all neighbouring regions.  
1. Select a region (node) to be the root node
2. Order nodes so that parents precede children
3. Backward pass: from end to beginning, check each node, and remove values from that node's parent that would violate our constraint (be same colour as parent). However, since there are 2 colours available for all regions, and since the problem is tree-structured, no values will actually be removed.
4. Forward pass: For each node from beginning to end, assign it a colour and remove any colours from its children that would violate our constraint (be same colour as parent).
